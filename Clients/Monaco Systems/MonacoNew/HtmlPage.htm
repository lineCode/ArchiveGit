Windows 95, the next version...

We're hard at work on the next version of Windows 95, code named "Memphis" which will be available in beta version the first quarter of 1997. Some of the new features in Memphis include:
  • Integrated Internet/intranet browsing with Internet Explorer in the box.
  • Built-in communications and collaboration with NetMeeting and Internet Mail and News.
  • Support for the latest hardware, including Universal Serial Bus (USB) and Digital Versatile Disk (DVD).
  • The latest multimedia components.
  • Features to make your PC more self-maintaining.
  • All features currently available only in Windows 95 OEM Service Release 2, including FAT32 support for large disk drives.
