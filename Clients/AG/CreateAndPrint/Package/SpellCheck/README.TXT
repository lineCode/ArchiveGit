Sentry Spelling-Checker Engine Windows SDK

Version 5.14.13 Patch
---------------------------------------------------------------------

This ZIP file contains the latest versions of the Sentry Windows DLLs
and dictionaries.

To use the patch, replace the corresponding files in the Sentry Windows
SDK (5.14 or later) with the enclosed files. See the Sentry Programmer's
guide for the locations of the files.

See http://www.wintertree-software.com/support/ssce/revisionhistory.html
for information on changes in this release.

The file "tech.tlx" contains a set of technical/computer/Internet words, such as
"HTML," "URL," "broadband," etc.
