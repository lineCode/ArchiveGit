Please allow the Create & Print ® software to install and run

Create & Print ® makes use of a specially developed software plug-in.  Our plug-in provides you with powerful features for personalizing and printing cards and other projects.  As we make improvements to our software, we pass them along to you as a plug-in update.  These updates allow you access to all of the improved features, and will not affect any other programs you may have.  In order to quickly install and run this software, see if one of the following conditions applies to you.

If there is a message above that reads:

This site might require the following ActiveX control: 'The Create & Print plug-in' from 'American Greetings, Inc.'  Click here to install...

then just click the message and choose the ' Install ActiveX Control...' option.  This will allow the Create & Print ® plug-in installation to proceed, and when successful, your card or project will appear here and you can begin to personalize it.

If you just answered 'Don't Install' or 'No' to a request to install or run the Create & Print ® plug-in:

then just click Refresh here, and answer 'Install' or 'Yes' when you are prompted again.  This will allow the Create & Print ® plug-in installation to proceed, and when successful, your card or project will appear here and you can begin to personalize it.

If there is any other message above related to ActiveX controls, or if you did not receive any prompt to install or run the Create & Print ® plug-in:

then one of your browser settings that controls plug-ins is probably disabled.  Just follow these easy step by step instructions to to enable the Create & Print ® plug-in.

  • Click on the Windows ® 'Start' button, RIGHT-click 'Internet Explorer', then click 'Internet Properties'.  The 'Internet Properties' window will pop up.
  • Click on the 'Security' tab in the 'Internet Properties' window.  This will display the security settings for the 'Internet' zone.
  • Click on the 'Default Level...' button.  Your security setting will be set to 'Medium'.
  • Click the 'OK' button to close the 'Internet Properties' window.
  • Click here to Refresh this page.  This will allow the Create & Print ® plug-in installation to proceed, and when successful, your card or project will appear here and you can begin to personalize it.

If you continue to have difficulty getting the plug-in to run, you may need to install the latest Internet Explorer components from Microsoft before using Create & Print ®.  For further information, check out our system requirements.
