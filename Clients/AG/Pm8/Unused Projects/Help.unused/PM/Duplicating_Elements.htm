Advanced Features and Tools:
 
Duplicating Elements

The Duplicate option is a quick and easy way to make a copy of an existing text box or picture. To make a duplicate of an existing text box or picture:

  1. With your mouse, select the text box or picture you want to duplicate by simply clicking on it.

  2. Click the Duplicate command in the Edit menu.

    The duplicate dialog box or picture appears.

  3. Choose the manner that you want PrintMaster to duplicate your element.

    Examples of each duplication method appear in the preview window.

  4. Click the OK button to create the duplicate element(s).

Design Note: You can manually reposition duplicates to create a variety of interesting three dimensional effects.
