Elements: Objects

Use the Object... command in the Add menu to place an element in your project created in another program. PrintMaster supports Object Linking and Embedding (OLE), which allows these objects to appear in your project but still be linked to the program used to create them. You modify the object using the program that created it.

To add an object to your project:

  1. Click the Object... command in the Add menu.

  2. Click the Create from file button.

  3. Click the Browse button to locate and select the file.

  4. Click the OK button, and the object appears in the workspace.

    Double-clicking on the file launches the program used to create it, and you can modify the object.

To create a NEW object to place in your project:

  1. Click the Object... command in the Add menu.

  2. Select the type of object from the list that appears.

  3. Click the OK button.

    The other program launches, and you can create the object. When you finish, a Save as Object... command appears in the File menu of the other program. After that, the object appears as an element in your project. Double-clicking on the file launches the program used to create it, and you can modify the object.
