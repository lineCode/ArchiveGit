Making Changes 

Select-then-do method:

  1. First, select the element(s)...

    When preparing to change any element - its size, position, color, or the words in a text box – the first step in the process is to select the element on the screen.

    How to select an element:
    - The simplest way to select elements is by clicking on them. Handles appear around any selected element. If you hold down the Shift key, you can click on several elements individually to select them at the same time.
     
    -You can also select an element (or a group of elements) with a “selection box”. Click in a blank area of the workspace, hold the mouse button down and drag the cursor until the selection box completely surrounds the element(s). (If you miss one, you won’t see the handles.)

  1. ... then, make your changes.

    After selecting an element, you can use any of the PrintMaster tools to change the way it appears.
