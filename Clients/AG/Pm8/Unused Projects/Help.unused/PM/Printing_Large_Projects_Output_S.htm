Printing Large Projects: Output Size

PrintMaster lets you take any project (e.g. cards, posters, banners) and print it at any size, from as small as a postage stamp to as large as a garage door. When you complete the project, you choose the size you want from within the Print dialog box.

  1. Click the Print button in the File toolbar, or the Print... command in the File menu.

  2. Click the Output Size... button.

    The Custom Print Size dialog will appear.

  3. Type the dimensions you want the printout to be in the boxes.

    If the printout is larger than a single sheet of paper it will print on multiple sheets—each sheet is called a tile. As you select the output size, click the Which Tiles... button to see a preview of the tile pattern in the final printout. When your project has finished printing, assemble the tiles to create the final printout.

    Note: PrintMaster maintains the relative proportions of the project. Changing one dimension will change the other dimensions proportionally.

  1. Click OK to confirm your selections, or click Cancel to return to the Print dialog without changing the output size.
