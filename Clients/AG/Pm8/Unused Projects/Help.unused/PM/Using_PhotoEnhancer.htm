Using PhotoEnhancer

PrintMaster includes PhotoEnhancer, an easy-to-use application to manipulate and edit your photos. Use PhotoEnhancer, to get rid of red eye, increase sharpness, adjust the brightness of your photos and change colors and textures of your graphic images.

To edit a picture using PhotoEnhancer:

  1. Click on an image.

  2. Click the PhotoEnhancer button in the Modify toolbar.

    The picture appears in the PhotoEnhancer window – ready for editing.

For help on using Photo Enhancer, click the Help command in PhotoEnhancer. The PrintMaster user manual also contains a Photo project tutorial that includes instructions for using some of the features of PhotoEnhancer.
