DESIGN BASICS:

 
Previewing and Printing...

Print Preview
You can view your project before printing by clicking the Print Preview button on the Global toolbar, or the Print Preview command in the File menu. In the Print Preview window, you can zoom in, zoom out, and print your project.

The design workspace displays the margins for your project as a gray box near the edges of the sheet. Only the parts of an element inside the margin will print, so the view that you see in the Print Preview window may not correspond to the view you have in the Project Window.

Printing your project:

  1. Click the Print button in the Global toolbar, or the Print command in the File menu. The Print dialog will appear.

  2. Confirm that the current settings are correct, then click the Print button to start the printing process.

    The Print dialog appears, allowing you to easily choose and configure your Windows printer, select the output size and print options for your project, and set number of copies to print.

The Print dialog:

(Click on the picture below for more information.)

Printer
The name and port of the selected Windows printer appears next to this item. To change printer, click the Setup button.

Print Quality
Most printers support a number of different print quality choices. Click on the Print Quality list box to make your selection. Generally, a low resolution or low quality setting will result in faster printing times—useful for printing a quick rough draft or a very large project that will be viewed from a distance. A high resolution or high quality setting will give you the best output, but will probably take longer to print—use high quality for your final printout.

Merge Names...
Use this option to merge fields from your Address Book onto your printed projects.

Output Size...
PrintMaster lets you take any project and print it at any size, from as small as a postage stamp to as large as a garage door.

Which Tiles...
For small printouts that fit onto a single tile (sheet of paper), the button is disabled. For large printouts that require multiple tiles, use this option to individually select the tiles to be printed.

Coloring Book
PrintMaster lets you print certain types of pictures as a picture outline - with all the colors replaced by white. The coloring book option can be set on a per picture basis, or from this dialog. Use this option on any picture included with PrintMaster.

Smoothing
Use the smoothing option to soften the roughness of low resolution bitmapped graphics. Smoothing affects bitmapped graphics format such as TIF or PCX, but not the pictures provided with PrintMaster (CGM).

Copies
Use this item to select the number of copies you want to print. Simply enter the desired number into the box next to the word Copies. The number of copies resets to 1 after each printout.
