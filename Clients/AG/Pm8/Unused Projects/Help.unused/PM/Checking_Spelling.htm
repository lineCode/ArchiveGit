Checking spelling

Click the Check Spelling button in the File toolbar, or the Check Spelling... command in the Tools menu to check for misspelled words in a selected text box.

  • If the Spell Checker finds a word it doesn’t recognize, it appears in the top box.

    The closest match for the unknown word appears highlighted in the box below – along with a list of other possible replacements.

  • Click the Change button to replace the unknown word with the highlighted word, or the replacement word you select from the list.

    Click the Change All button to repeat the change anywhere else it occurs.

  • If you use a word that you know is spelled correctly, but the spell checker doesn’t recognize it, click the Add button to include it in the dictionary.

    Use this method to avoid unnecessary time spent on technical terms, slang phrases, words from another language, and so on.

  • Check the Ignore boxes to skip over all uppercase words, or words containing numbers.
