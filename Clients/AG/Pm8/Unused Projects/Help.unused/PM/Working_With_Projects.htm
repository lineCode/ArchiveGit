WORKING WITH PROJECTS

 
Working with Your Projects

Clicking the Open command in the File menu opens the Project Gallery browser - where you can continue working on a project, or select a new project. PrintMaster also keeps a list of links to the last four projects you worked on at the bottom of the File menu. Clicking on these links opens the recent project.

PrintMaster saves your projects in a separate folder on your hard drive. In the Project Gallery browser, you can always search out the projects you've created by choosing the Your Own category in the Project Type list.

Category:
PrintMaster also lets you save your projects into your own groups--called "categories". When you save a project for the first time, or click the Save As command in the File menu of the main screen, you'll see this option at the bottom of the dialog box. Choose one of your existing categories from the list, or click the New... button to create a new category.

To open one of your projects...

  1. Click the Open command in the File menu.

  2. Click the arrow beside the list of Project Types and choose “Your Own Projects” as the category.

    Small images of your projects appear in the right side of the Project Browser window.

  3. Click on the project you wish to open, then click the Select button.

Opening projects from disk.
If the project you want does not appear in the Project Gallery browser, you may need to open it from a disk (floppy disk or hard disk). In the Project Gallery browser, click the Open from Disk... command in the File menu to browse a disk for the project you want.
