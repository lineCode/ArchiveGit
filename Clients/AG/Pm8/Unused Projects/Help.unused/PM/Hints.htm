GETTING STARTED:

 
Hints:

  • You’ll need the PrintMaster CD...

    In order to start PrintMaster, you must insert the CD into the CD-ROM drive. However, once the program is up and running you may remove the CD for brief periods in order to access graphics from other CDs.

  • The invisible button...

    Sometimes you will see that a button in a toolbar, or a command in a menu does not seem to work, or appears fuzzy or gray in color. These commands and buttons are not available because of the element you are working with. Each type of element (text, picture, etc.) has its own group of menu commands and toolbar buttons, so only the ones you might need are available. If the command or button you want to use is not available, check to see that you have the right element selected.

  • Use your right mouse button...

    Any time you need to modify an element on the screen (e.g. line, picture, text) put the cursor arrow over the object, then click the right mouse button for instant access to an appropriate set of tools and commands for modifying that type of element.

  • Anything you can do, you can undo...

    Any time something unexpected happens, and you need to go back to where you were, click the Undo command in the Edit menu, or the Undo button in the ‘Global’ toolbar. You can undo almost any action in this manner.

  • Helpful Hints...

    From time to time you will see a small message window appear with a tip, trick or reminder to guide you as you learn to use PrintMaster.

    To see the most recent Hints:
    1) Click the Recent Hints command in the Help Menu.
    2) Click on the Hint you want to see.
    3) Click the Show Hint button.

    To hide or reset the Helpful Hints:
    1) Click the Preferences command in File menu.
    2) Click the Hints tab.
    3) Select the option(s) you want to modify.
