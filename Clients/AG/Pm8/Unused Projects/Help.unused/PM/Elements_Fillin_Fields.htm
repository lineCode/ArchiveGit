Elements: Fill-in Fields

To add a fill-in field to your project:

  1. Create a blank text box by selecting 'Text Box' from the Add menu, or by clicking the Text Box icon on the Edit Bar.

    NOTE: Fill-in fields do not have to be inserted into an empty text box. They can also be added to text boxes that already contain existing text. Mail Merge fields are inserted at the text cursor position.

  1. Select Fill-in Fields from the Tools menu, then select the 'Add Fill-in field' command.

    A list of available field names will appear on the left side of the Add Field dialog box. Select a field and press 'Insert'. You can repeat this process as many times as necessary.

  • To add a custom field, click the Add button in the Add Field dialog box and enter a name for your new field.

  • To automatically merge information from the Receivers List of your Address Book, click the Receiver... button. The Select Merge Names command in the Tools menu takes you to the Receiver's Address Book.

  • Pick the receiver name(s) you wish to use by clicking on them with your mouse, or press the Select All button to include all of them.

  1. Once you have chosen the desired name(s), click OK.

    Note: When selecting multiple receiver names, only one of the chosen names will appear on the project page (generally the first one selected). You will also only see one name listed in Print Preview. However, upon printing your project each printed page will contain a different receiver name.

    The fill-in fields seen earlier have been replaced by the first name in the receiver list you selected. At this point, you can begin editing the way your text will look by changing fonts, font color and size, and other properties.

  2. When you are done, click 'OK'. The fill-in field is placed on your project where you positioned your text box, and can be moved and sized just like a text box.

To modify a fill-in field:
Once you have placed fill-in fields onto your project page you can quickly change the values in each of the inserted fields.

  1. Select Fill-in fields from the Tools menu, then select 'Edit Fill-in fields'.

    All the fields in your project appear in the Fields dialog box.

  2. Use Tab and Shift+Tab to move to the next field or to back up to a prior field. You can change what each field will say by entering text in the 'Fill-in Value' box: this text will appear on the project in place of the standard text. (For example, to address members of a bowling league, you might first add the fill-in field labeled 'Company' then edit it by entering 'League' or 'Team' in the Fill-in Value box.)

    Note: You can also select values from the Address Book so your Address Book information merges automatically.

To remove a fill-in field:
Remove a fill-in field by simply highlighting it with your mouse and pressing the Delete key.
