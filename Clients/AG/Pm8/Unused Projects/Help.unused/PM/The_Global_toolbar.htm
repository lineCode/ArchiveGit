The File toolbar

The File toolbar appears at the top of the Workspace, providing quick access to some of the most commonly used PrintMaster actions such as closing the current project, going to the Ready Made Gallery, starting a With Help project, and starting a Brand New project. These buttons are one-step alternatives to using the menu bar for Saving, Printing, Cutting, Pasting, and others.

New Button
Click this button to display the "Hub" screen where you can begin a new project, open an existing project, etc.

Save
Click this button to save the current project.

Print
Click this button to send your project to your printer.

Print Preview
Click this button to see how your project will print out – before you print.

Check Spelling
Click this button to check the spelling of the text in your project. A text box must be active to use this function.

Cut
Click this button to remove the selected text and/or graphic, and copy it to the Windows Clipboard. Use the Paste function to restore elements you “Cut” into any open project.

Copy
Click this button to copy the selected text and/or graphic to the Windows Clipboard. Using the Paste function, you can duplicate elements you “Copy” into any open project.

Paste
Click this button to insert a copy of the Windows Clipboard contents into your project.

Undo
Click this button to quickly undo changes in editing, formatting, and other actions. To quickly undo multiple actions, click this button more than once.

Redo
Click this button to reverse changes made using the Undo button.

 

Note: To hide or display the File toolbar, click the Toolbars command in the View menu, then click on the 'File Bar'. Check marks show which toolbars will appear on screen. Follow this procedure to display or hide any toolbar.
