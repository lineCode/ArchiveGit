Web Page Links

Adding Hyperlinks and Hotspots:
By assigning hyperlinks to your web page objects, you can move your readers quickly to relevant text, graphics, files, or other web pages. You can also add invisible hyperlinked 'hotspots' over any portion of a picture or project page.

To add a hyperlink to your web page:

  1. Select the text or object in the Workspace you want to turn into a hyperlink.

  2. Then, select Hyperlink... from the Web menu.

To add an invisible hotspot over any area of your web publication:

  1. Select Create a Hotspot from the Web menu.

  2. Drag the mouse over your web publication to create the hotspot.

    You will then see the following Hyperlink dialog. Here you can choose where the hyperlink or hotspot will take your viewer when they click on it.
