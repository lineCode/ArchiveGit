GETTING STARTED:

 
Getting Started

  

Beginning Computer Users:
If you are a new user, browse through this Help file to get a basic understanding of PrintMaster. In addition, the user manual contains several tutorials that will guide you step-by-step through the creation of a sample project.

Undo:
Remember that you can almost always un-do anything you do in PrintMaster. The “Undo” button appears in the Global toolbar, and at the top of the Edit menu.

Tool tips:
If you forget what button to click when you want to do something, just use the mouse to move the cursor arrow over any button. After a second, the name of the button appears to help remind you what the button does.

Experienced Computer Users:
As you become familiar with PrintMaster, you’ll discover that there is generally more than one way to do the same thing. Experimenting with different methods will help you find the most comfortable method for you.

For example, if you select a few words of text in a text box you could make them bold in any of four ways:

  1. Click the Bold (B) button in the Text toolbar.

  2. Hold down the Ctrl key and then press the B key on the keyboard (also known as the Ctrl + B shortcut or “hot key” combination).

  3. Click the Font... command in the Format menu, then click Bold in the Font Style box.

  4. Click the right mouse button beside the selected words to display the popup menu, then click the Bold command.

Advanced Computer Users:
If your experience qualifies you as a professional desktop publisher, or if you have experience using graphic and desktop publishing software, PrintMaster contains many sophisticated tools and options beneath the surface. The intuitive interface and familiarity of the functions and commands will get you from concept to publication in no time. The more you look, the more options you’ll find to do exactly what you want.
