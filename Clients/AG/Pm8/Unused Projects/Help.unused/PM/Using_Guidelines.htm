Arranging Elements:
Using Guidelines

Guidelines allow you to control the position of elements on the page. Placing a guideline on your page lets you align elements to the same horizontal or vertical position on the page.

Adding a guideline
To create a guideline, position your point over the horizontal or vertical ruler. Press the left mouse button and drag a ruler guide to any position on the page. You can also use the menu by selecting Add Guide (Vertical or Horizontal) from the Arrange menu.

Removing a guideline
To remove a guideline, drag it back to the ruler.

Snap to Guidelines setting
Guidelines can be set to act like magnets on the page - so when you move an element near it, the element will "snap" into position against the guideline.

Lock Guides setting
When you have a guidline set exactly where you want it, click the Lock Guides command in the Arrange menu. You won't be able to select or move the guidelines on the page accidentally when you check this option.
