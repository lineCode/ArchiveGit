Printing a Project Catalog

You can print miniature copies of a group of projects as a reference guide from the Project Gallery. When working with several projects, you can create a catalog for easy reference. Use the catalog to find the project you want.

To print a project catalog:

  1. Click the Print... command in the File menu.

  2. Select the Print Range.

  3. Select the number of images you want to appear on a single page.

    PrintMaster will scale them automatically to fit the page - regardless of their actual size.

  4. Select the print quality and other options if necessary, then click the Print button.

    The number of pages will depend on the Print Range, and the number of images per page.
