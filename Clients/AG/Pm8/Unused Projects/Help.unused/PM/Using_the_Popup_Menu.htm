Changing and Arranging Elements:
Using the Popup Menu

The popup menu is a quick way to access common tools and options. It appears whenever you click the right mouse button while the cursor is over an element in the workspace area.

The tool and command options available in the popup menu depend on the type of element and its position. Text boxes have one set of options (shown here), and graphic elements (e.g. pictures, shapes, lines) have another set of options.
