Working with Images

PrintMaster creates a “Your Own” collection for any images you open or import from other sources. PrintMaster copies the files into a separate folder in the PrintMaster program directory on your computer’s hard drive.

To add a picture you have on disk to your project:

  1. Open the Art Gallery browser by clicking on the Picture button in the Edit toolbar.

  2. Once you are in the Art Gallery, click the Open from Disk... command in the File menu. A dialog box appears for you to select your file. Navigate to the appropriate location (floppy disk drive or directory).

  3. Select the file, then click the OK button. You picture will appear in the middle of your project in the Workspace.

Note: You can also use the Import command in the File menu to add groups of image files to ‘Your Own’ category in the Art Browser.
