Applying Values to Fill-in Fields

Once you have placed fill-in fields onto your project page you will need to assign values to each of the inserted fields.

  1. To begin, you must first create a blank text box by selecting 'Text Box' from the Add menu, or by clicking the Text Box icon on the Edit Bar. Select Fill-in Fields from the Tools menu, then select the 'Add Fill-in field' command.

    A list of available field names will appear on the left side of the Add Field dialog box. Select a field and press 'Insert'. You can repeat this process as many times as necessary.

    To add a custom field, click the Add button in the Add Field dialog box and enter a name for your new field.

    When you are done, click 'OK'. The fill-in field is placed on your project where you positioned your text box, and can be moved and sized just like a text box.

  2. To edit a field, again select Fill-in fields from the Tools menu, then select 'Edit Fill-in fields'. Your fields appear in the Fields dialog box.

    Use Tab and Shift+Tab to move to the next field or to back up to a prior field. You can determine what each field will say by entering text in the 'Fill-in Value' box: this text will appear on the project in place of the standard text. (For example, to address members of a bowling league, you might first add the fill-in field labeled 'Company' then edit it by entering 'League' or 'Team' in the Fill-in Value box.)

    You can also select values from the Address Book so your Address Book information merges automatically.

    To automatically merge information from the Receivers List of your Address Book, click the Receiver... button. The Select Merge Names command in the Tools menu takes you to the Receiver's Address Book.

  3. Pick the receiver name(s) you wish to use by clicking on them with your mouse, or press the Select All button to include all of them.

  4. Once you have chosen the desired name(s), click OK.

    Note: When selecting multiple receiver names, only one of the chosen names will appear on the project page (generally the first one selected). You will also only see one name listed in Print Preview. However, upon printing your project each printed page will contain a different receiver name.

    The fill-in fields seen earlier have been replaced by the first name in the receiver list you selected. At this point, you can begin editing the way your text will look by changing fonts, font color and size, and other properties.
