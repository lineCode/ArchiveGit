Creating Matching Sets

With Matching Sets, you can create a look for your business, organization, or event effortlessly. A matching set might include letterhead, business cards, brochures, newsletters, and envelopes for your business.

Matching sets are projects that look similar. These projects use the same colors, follow the same theme, or share the same images. They look like they belong together. The PrintMaster Project Gallery contains many ready-made projects that combine to form matching sets. Use these projects as the starting point for creating your own matching sets, or create them on your own.

To choose a project from the ready-made Matching Sets:

  1. Open the Project Gallery:

    • In the Hub screen, click the Ready-Made button, or the Open... command in the File menu.

    • In the Design Workspace, click the Ready-Made button in the Global toolbar, or the Open... command in the File menu.

  2. In the Project Type list, click Matching Sets.

  3. Select the kind of Matching Set from the project window on the right.

  4. Click on a project, then click the Select button to open it.

    From the Design Workspace, you can repeat these steps to open more than one project in the matching set at a time.

Hint: Save matching sets projects together so they can share the same images – when you edit the image, it will transfer the changes to all the projects.
