Web Publishing

The Design Checker:
Once you have completed your web page, use PrintMaster's Design Checker to review any formatting issues that may cause problems when the page is displayed online.

  • To run the Design Checker, select Design Checker under the Web menu.

NOTE: PrintMaster's Design Checker will also be launched when you publish or preview your web page - unless you uncheck the box in the Preferences dialog.

Previewing Your Web Publication:
It's a good idea to preview your web site before publishing it online. By previewing it in a browser first, you can be sure the text is readable, the graphics appear properly, and the hyperlinks and hotspots work correctly.

  • To preview your web publication, select Preview Web Documents from the Web menu.

Publish your Web Site:
To publish your web site as an HTML file to the Internet, select Publish to Web Site under the Web menu. You can also publish your web site to a folder on your hard drive by selecting Publish to Folder.

NOTE: Publishing to the Web can be confusing. Before publishing, you will have to have an Internet connection and an agreement with an Internet Service Provider (ISP) allowing you to publish your web site on their server. You may also need to follow the ISP's file naming conventions and use their installation software before publishing your site.
