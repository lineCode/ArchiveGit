Printing a Picture Catalog

You can print miniature copies of a group of pictures as a reference guide from the Art Gallery. When you work with large numbers of images, or want to create a ready reference for the graphics you have, printing a picture catalog is an easy solution. Catalogs help you find the picture you want in less time than it takes to open images one by one.

To print a Picture Catalog:

  1. Open the Art Gallery by clicking the Art Gallery button in the Edit toolbar, the 'Picture' command in the Add menu, or clicking the Art Gallery button in the Hub screen.

  2. Click the Print... command in the File menu.

  3. Navigate to the directory, and select all the pictures to include in the catalog.

    Press the Shift or Ctrl keys when you click on the files to select multiple images. (You might want to move all the images into the same directory first, to avoid having to print several catalogs.)

  4. Click the Print button to open the Print dialog.

  5. Select the number of images you want to appear on a single page.

    PrintMaster will scale them automatically to fit the page - regardless of their actual size.

  6. Select the print quality and other options if necessary, then click the Print button.

    The number of pages will depend on the number of images selected, and the number of images per page.
