Design Basics:

  Making Changes:

The Edit Toolbar

Use the Edit toolbar to make adjustments to your art and text. With these tools, you can easily change the color of text and pictures, change line widths, add shadows, and crop pictures to see only as much of it as you want.

PhotoEnhancer
PhotoEnhancer is an easy-to-use image editing and manipulation application included in PrintMaster. Just select the image you want to modify, then click this button.

Fill color
Click on this tool to add color to your elements. First, select the element you want to color, then select this tool. A selection of colors appears to choose from. If you select a rectangle, ellipse or shape, you will also see a Gradient button which allows you to add a blend of colors to these elements.

Line format
Use this tool to change the line width of your drawing or the outside border of a shape.

Shadow format
Use this tool to add a shadow to your text or shapes. The choices vary for text and shapes.

Crop
Use this tool to crop out the part of a picture or shape you do not want to see.

 

Note: To hide or display the Edit toolbar, click the Toolbars command in the View menu, then click on the 'Edit Bar'. Check marks show which toolbars will appear on screen. Follow this procedure to display or hide any toolbar.
