Using Ready-Made Designs

Ready-Made projects allow you to quickly customize a design to suit your needs. They can be a real time-saver if you are just learning to use PrintMaster, or any time you need a job done fast!

Clicking the Ready-Made button in the Hub screen opens a "browser" window where you can select a pre-designed project to use. Small preview images appear in the right side of the window that correspond to the descriptions in the left side.

Use the lists to narrow down the number of choices until you find one that you like, or type in a couple of descriptive "keywords" to search for a project.
