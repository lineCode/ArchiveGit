Adding Design Elements:

Text Wrap

Text boxes have the ability to cover elements below them, or adapt the position of the letters to avoid covering any other elements. This property is known as "text wrap".

When a text box is selected, the Text wrap command appears in the Arrange menu. Clicking this command displays two choices for the text box:

Text Flows Over:
This option ensures the text box stays above all the other elements around it.

Text Wraps Around:
This option causes the words in the text box to adjust their position to create space around any element above or below the box. The text “wraps” around these other elements.
