Arranging Elements:
Using Layers

A project is like a stack of transparent layers piled on top of each other. In PrintMaster, these layers fill the entire workspace. You can see through any space in a layer that is not filled by an element. Each layer can have one element on it or it can have a “group” of elements.

When you add an element (a text box, line, shape, or picture), you add a new layer to the project. Each layer has its own place in the stack. An element on one layer will cover an element on a layer directly below. Elements not directly below are still visible, because the layers are transparent.

When you click on an element to select it, you are selecting the entire layer. Use the popup menu, or the Arrange menu to bring an element in front of another. By changing the order of an element’s layer in the stack, you can place elements in front, back, or anywhere in between other elements.
