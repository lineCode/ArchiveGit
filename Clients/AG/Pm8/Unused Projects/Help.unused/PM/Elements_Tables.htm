Adding Design Elements:
Elements: Tables

A table is a small grid that can be placed on your project for arranging many small bits of text information or aligning several small pictures. You can format the table, modify the columns and rows, and merge cells to create grids of all types and colors. The table, and the text typed in it, can be moved and adjusted just like any other PrintMaster object.

To create a table object:

  1. Click the Table Tool and drag across your project page. The Table Format dialog will appear.

  2. Enter a value from 1-30 in the number of rows and number of columns text boxes.

  3. Scroll through the list of table formats and select one.

  4. Click OK.

    You can now insert text into any of the cells by clicking in the cell and typing. To move the cursor to the next cell, press the Tab key.

Fill Down:
You can repeat text in every cell of a selected column by selecting Fill Down... in the Table menu. The text in the top selected cell will be copied down.

Adjusting the size of columns and rows:
To adjust the width or height of your columns and rows, place the cursor on the column or row border. The cursor will turn into an 'adjust' cursor. You can now drag the column or row border to adjust it. To keep the whole table the same size while only changing one column or row, hold down the Shift key while dragging the column or row border.

To insert additional columns or rows:

  • Select the whole column beside the point where you want to insert a new column, then click the Insert command in the Table menu.

  • Select a whole row above or below the point where you want to insert a new column, then click the Insert command in the Table menu.

    The Insert dialog box appears to let you choose how many rows or columns to add and wher to put them into the table.
