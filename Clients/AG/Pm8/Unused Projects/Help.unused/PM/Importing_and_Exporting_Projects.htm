WORKING WITH PROJECTS:

 
Importing and Exporting Projects

PrintMaster creates a “Your Own” category for any project you open or import from other sources. PrintMaster copies the files into a separate folder in the PrintMaster program directory on your computer’s hard drive. Adding a project from an outside source to your computer is known as “importing”; sending a project to a disk or another computer is known as “exporting”.

PrintMaster can also export the projects you create in graphic file formats that are compatible with other graphics and publishing programs.

To export a project in another file format:

  1. Click the Export As... command in the File menu.

  2. Select the disk and directory where you’d like to copy the file.

  3. Choose the file format for the file you will create.

    PrintMaster can export files in BMP, JPG, PNG, GIF, PCX, TIF, and WMF formats. You can also choose to add the graphic to the Art Gallery.

  4. Click the Save button.

To import files:

  1. Click the Open... command in the File menu to open the Project Browser.

  2. Click the Import... command in the File menu of the Project Browser.

  3. Navigate to the disk and directory where the project resides and select the file.

    Category:
    PrintMaster also lets you save your projects into your own groups--called "categories". Choose one of your existing categories from the list, or click the New... button to create a new category.

  4. Click the Import button.

    PrintMaster adds the project to your list of projects, and a small image of the project appears in the right side of the Project Browser window.

To export files to disk:

  1. Click the Open... command in the File menu to open the Project Browser.

  2. Click the Export... command in the File menu of the Project Browser.

  3. Select the project to export from the file list.

  4. Select the disk and directory where you’d like to copy the file.

  5. Click the Save button.
