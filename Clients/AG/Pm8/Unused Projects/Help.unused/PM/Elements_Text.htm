Adding Design Elements:
Elements: Text

PrintMaster gives you the power to create text in almost any font, size, shape and color you can imagine. If you are familiar with word processing software, working with text in PrintMaster will seem very familiar.

The text box:
All text in PrintMaster projects appears inside a rectangular area also known as a “text box” (sometimes referred to as a “frame”). A text box is a single element containing as many words as you like. As with other elements, you can click on the text box and move it anywhere on the page.

The text in the box stays together as a unit, but you can create any number of text boxes. If you click inside the text box, you can edit the text as with any standard word processor (think of text boxes as miniature document windows).
