DESIGN BASICS:

Adding Design Elements:

Text-Formatting

The Format menu and the popup menu provide access to all the style, shape and visual effects you can apply to text. This includes several advanced commands that are not accessible from the toolbars that can add impact to your project.

Scroll down this screen to find information about the text-related commands in the Format menu:

Format > Font:

Clicking the Font command in the Format menu displays a box with two pages (“tabs”) of font selections:

The Font tab:

This screen lets you select or modify the Font, Font Style, Font Size and Color. The Preview window at the bottom of the screen displays some sample text with the settings you select.

The Effects tab:

This screen lets you add special effects to any text you type or select. The Preview window at the bottom of the screen displays some sample text with the settings you select. (These effects may be hard to see on small text.)

  • Fill Pattern and Color

    These settings apply a pattern to the letters of the text you select, using the color you specify. Click the More... button to choose any color not shown in the list.

  • Outline and Shadow

    These settings allow you to select from several special visual effects, in any color you choose.

Format > Character Spacing:

Clicking the Character Spacing command in the Format menu lets you select how much space there is between the letters and words in a text box.

Each font has a fixed distance for each character on the keyboard, and for the spaces between words. By changing the character spacing setting in this box, you can make more or less text appear on each line – depending on your project. A sample appears in the Preview window at the bottom of the screen.

Format > Paragraph:

Clicking the Paragraph command in the Format menu displays a group of settings related to paragraphs as a whole – rather than to individual words.

  • Indentation:

    Use this setting to set the minimum distance between the text and the edge of the text box (also known as the “margin”) on the left and right sides. You can also set the indentation for the first line of a paragraph separately from the other lines.

  • Line Spacing:

    This setting determines the spacing between lines. The default is “1 Line” – known as “single spacing” - but you may change the number as needed (2 Lines means “double spacing, and so on).
     
    Changing the units from Lines to Points lets you select a distance between lines that is less than one line or equal to a unique value. “Points” refers to the size of the font, so if your text is 12 points in size, then setting the Line Spacing to “6 Points” would be equal to half a line spacing, and so on.

  • Alignment:

    This setting aligns the text in the paragraph to the left margin, right margin, center of the margins, or spread between the margins (“justified”).

  • Paragraph Spacing:

    This setting allows you to add extra space between paragraphs, rather than between lines of a paragraph. For example, a paragraph may be single spaced (1 Line), but have an extra 6 points after the last line (when you press the Enter key to start a new paragraph). This separation can prepare your reader for the message in the next paragraph by providing a visual break between it and the previous paragraph.

Format > Tabs:

Clicking the Tabs command in the Format menu lets you set the amount of space between words when you press the Tab key on the keyboard. The left side of the window lets you set custom distance(s), and the right side shows the current and overall tab settings.

  • Default Tab Stops:

    This value places “markers” at equal distances along any line (“tab stops”). When you press the Tab key, the cursor moves to the next tab stop down the line. Tab stops can help you line up words in columns, and changing the setting affects all text boxes.

  • Tab Stops list:

    This is a list of any tab stops in the active paragraph (if any). When you create a custom tab stop, it appears in this list. To remove a tab stop from the list, click on it, then click the Clear button.

  • Position:

    The first thing you do when setting a custom tab is to specify its distance from the left margin. This is the position of the tab stop. After you type the position, you can click the Set button to create the tab stop.

  • Alignment:

    Each tab stop you create can position the next words in the line to the right, left or center of the tab stop you create. (You can also specify a decimal distance.)

  • Leader:

    When you press the Tab key in a paragraph, you can specify that the space created between words be filled with a solid line, nothing, or a custom character (an asterisk or other character). This “leader” is useful in helping your reader see the connection between words in a line. (For example, imagine the Table of Contents or Index of a book that uses a dotted line between the page number and the topic.)

Format > Text Box:

The Text Box Formatting window contains four settings that affect the shape and appearance of the words in a text box. These settings affect a text box as a whole.

  • Margins:

    These four values set the distance between the words and the edge of the text box where they appear.

  • Columns:

    This setting divides the paragraph evenly into the Number of columns you set - separated by the amount of Spacing you specify.

  • Vertical Alignment:

    This setting aligns the text in the text box to the top, middle or bottom of the text box.

  • Shape:

    This setting alters the shape of the text in a text box. You can choose from many shapes to make your words stand out.
