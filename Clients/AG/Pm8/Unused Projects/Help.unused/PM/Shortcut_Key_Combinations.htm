Advanced Features and Tools
 
Shortcut Key Combinations

The following is a list of keyboard buttons and combinations of buttons that perform certain tasks in PrintMaster without the need of a mouse or menu command. Use these shortcuts or “hot keys” to save time searching for buttons or menu commands.

General Editing

Ctrl-A Edit/Select All

Ctrl-X Edit/Cut

Ctrl-C Edit/Copy

Ctrl-V Edit/Paste

Shift-Del Edit/Cut

Ctrl-Ins Edit/Copy

Shift-Ins Edit/Paste

Ctrl-N File/New

Ctrl-O File/Open

Ctrl-S File/Save

Ctrl-P File/Print

Del Edit/Delete

Alt-Backspace Edit/Undo

Ctrl-Z Edit/Undo

Ctrl-Y Edit/Redo

Plus (+) key Zoom in

Minus (-) key Zoom out

F1 Help

F6 Next window

Shift-F6 Previous window

When an Element appears selected:

Left arw Move object 1 pixel left

Right arw Move object 1 pixel right

Up arw Move object 1 pixel up

Down arw Move object 1 pixel down

Address Book:

Ins Add New Name

Del Delete Current Name

Home First Name

End Last Name

PgUp Previous Name

PgDn Next Name

Text Editing:

Ctrl-B Toggle bold text

Ctrl-I Toggle italic text

Ctrl-U Toggle underline text

Left arw Move cursor left (*)

Right arw Move cursor right (*)

Up arw Move cursor up (*)

Down arw Move cursor down (*)

Ctrl-left arw Left a word (*)

Ctrl-right arw Right a word (*)

Home Beginning of line (*)

End End of line (*)

Ctrl-Home Beginning of text (*)

Ctrl-End End of text (*)

(*) Hold the Shift key to create or extend a selection.
