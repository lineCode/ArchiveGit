Fun For Kids

Fun For Kids lets kids design, print and play with masks, puppets, stickers, bookmarks and more!

To find projects for kids:

  1. Open the Project Gallery by clicking the "Ready Made" button on the hub screen, or the Open... command in the File menu.

  2. Select "All" from the Project Collection list.

  3. Select "Fun For Kids" from the Project Type list.

  4. Click the Select button.
