Arranging Elements:

 
Tools for Arranging Elements

The Format toolbar contains tools for working with all objects in PrintMaster. With these tools, you can easily select pictures partially hidden behind another, group together objects you want to move as one, or flip pictures horizontally and vertically. To hide or display the Place toolbar, access the View menu, then Toolbars/Place Bar.
