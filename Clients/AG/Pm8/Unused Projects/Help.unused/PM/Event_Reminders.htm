Advanced Features and Tools
 
Event Reminders

PrintMaster automatically installs a small application that monitors your address book and notifies you of any birthday, anniversary or other event ahead of time - so you won’t miss another important date.

The Event reminder application contains some preset options, and guides you step-by-step in selecting the types of events to notify you about.

To view the Event Reminder settings, or set up your own reminders, click the Event Reminder... command in the Tools menu and follow the instructions on screen.

Opening Screen:

When Event Reminder launches, the next or nearest event appears in the middle of the first screen.

  • To view, add or change the list of events for which you will see notifications, click the Event Reminder List... button.

Event Reminder List:

You’ll be notified about these events. Click the Delete button to remove an event type from the list. Click the Change... button to change the settings for the event type. Click the New... button to add another reminder.

The Reminder Wizard:

Click the New... or Change... buttons opens the Event Reminder Wizard. Select or confirm the Address book to use, the type of event, the name of the person or persons to track, and how far ahead to notify you before the event. When you finish the new or revised item appears in the list.
