DESIGN BASICS:

 
Arranging Elements

With PrintMaster, you can move any element on the screen by simply clicking on it, positioning the cursor so that the “move icon” (the four-way arrow) appears, and holding down the mouse button while dragging the item. The positions of the elements on the screen are the same as they will appear when you print the project.

Advanced Techniques:
PrintMaster has many other ways to help you position elements just the way you want. For example, you can easily select pictures partially hidden behind another, group together objects you want to move as one, or flip pictures horizontally and vertically.

Some of these commands appear as buttons in the Format toolbar or as commands in the popup menus, but all appear in the Arrange Menu.

Click on the menu to the left for descriptions of the commands in the Arrange menu.

Arrange / Position>:

Center on Page
This tool centers the active object in the middle (half way between the top and bottom and left and right sides) of the project.

Center Horizontally / Vertically
These tools center the active object in the middle of the project either from top to bottom (vertically), or from left to right (horizontally).

Full Page
This tool stretches the active object to fill the lenght of the project window.

More...
This tool opens a dialog box where you can precisely set the size and position of the active object.

Arrange / Layer>:

Bring to Front
Click this tool to bring the active object to the top level of a stack of overlapping objects.

Forward One Layer
Click this tool once to bring the active object up one level.

Back One Layer
Click this tool once to push the active object back one level.

Send to Back
Click this tool to send the active object to the bottom level of a stack of overlapping objects.

Arrange / Previous/Next Object:

Select Next Object
Click this button to select the object in the next layer.

Select Previous Object
Click this button to select the object in the previous layer.

Arrange / Mirror/Flip:

Mirror
This command flips the active element horizontally.

Flip
This command flips the active element vertically.

Arrange / Text Wrap...:

Text Flows Over
Choosing this option causes text characters to cover elements on layers beneath them.

Text Wraps Around
Choosing this option causes text characters to adjust their position so the won't cover elements on layers beneath them. The text fills the space around the other elements.

Arrange / Group/Ungroup:

Group
Use this command to keep several elements together. Holding down the Shift key, click on all the elements one by one, then click the Group command. The grouped objects will now act as one large element.

Ungroup
Click this tool to ungroup several objects you have previously grouped.

Arrange / Settings:

Arrange Settings
Click on these settings to control the behavior of elements when you arrange them on a page, or to create guidelines to place elements where you want them.
