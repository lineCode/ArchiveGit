Creating Web Pages

With PrintMaster, you can create beautiful web pages without knowing a word of programming language! Each page of your web 'site' can include graphics, photos, and text, as well as links to sound files, digital videos, and other web sites.

After you create your web pages in PrintMaster, PrintMaster will convert them to the format required by the Internet-Hypertext Markup Language (HTML)-then publish them to the World Wide Web.

You can create web pages just like any other project - choose one of PrintMaster's professionally-designed Web sites, or design your own from scratch. The Ready-Made Web sites are multi-page templates that include graphics, pre-designed text, hyperlinks, and animations. You can add images, text, and borders to your Web publication, as you would in any other PrintMaster project.
