Advanced Features and Tools
 
Setting Preferences

Clicking the Preferences command in the File menu will display the Preferences dialog box where you will find several ways to customize your working environment.

Sound Effects:
To turn on or off the sound effects that accompany many mouse clicks:

  1. Select the Sound Effects tab at the top of the Preferences dialog.

  2. Select On or Off.

Hints:
PrintMaster includes two types of "hints"-pop-up windows with text information-that can guide you through your projects. Helpful Hints present useful information about creating projects, aligning pictures, and introduce new areas of PrintMaster. Design Tips are professional tips for creating beautiful and well-designed projects.

To turn off or reset either of the hint windows:

  1. Select the Hint tab at the top of the Preferences dialog.

  2. Select or deselect the check boxes for each type of hint.

    After most hints are presented once or twice, they will not show up again (unless you select them from the Recent Hints menu). If you would like to reset all of the hints so you can see all of them again, click Reset All Hints.

Workspace Preferences
Since your hard disk is faster than your CD-ROM player, PrintMaster improves performance by using a file on your hard disk to store the most frequently used information. This hard disk file is called the workspace file.

Type in a new workspace size to change this value.

  • A larger workspace uses more hard disk space but enables you to access fonts and graphics far more quickly.

  • A smaller workspace uses less hard disk space but takes longer to access the fonts and graphics.

Online:
In order to access the Online Art Gallery or send an Online Greeting, you will need to connect to the Internet through a modem or direct connection. Select the Online tab in the Preferences dialog to select which type of connection you will be using. If you are using a dial-up modem connection, select Properties... to properly configure your modem connection properties and select the browser you will be using.

If you do not want to run the design checker when you publish your web pages, uncheck the box that says "Use Design Checker when Publishing Web Pages".
